Each chandelier has 3 motorized rings. Per ring: 7 motor channels + 10 LED channels. Plus 3 globe dimmers.

Motor Channels (7 per ring):
  • SPEED (1ch): Motor speed. 0 = brake ON, >0 = brake OFF + move
  • TILT (2ch): NOT USED — fixture profile only
  • TURN (2ch): NOT USED — fixture profile only
  • DROP (2ch): HEIGHT CONTROL. 0 = up, 255 = down

Note: Rings are winch-driven (3 cables each). They move up/down only — no tilt or rotation capability.

LED Channels (10 per ring): R, G, B, W, Dimmer × 2 strips. Dimmer must be >0 for output!
